## 一、公司与品牌背景

### 1.1 公司简介

VanGATE Manufacturing and Automation Inc. 是一家位于加拿大不列颠哥伦比亚省的**全垂直整合门禁与自动化解决方案制造商**，提供从工程设计到制造、安装和维护的一站式服务。

**核心优势：**
- 加拿大西部唯一真正的垂直整合制造商
- 自有工程设计、CNC加工、焊接制造能力
- 覆盖住宅、商业和工业全场景

**核心产品线：**
| 类别 | 产品 |
| --- | --- |
| 住宅产品 | 车道门、车库门、电梯、门禁自动化、维修服务 |
| 商业/工业产品 | 安全门、工业门、屏障系统、门禁控制、维保服务 |
| 定制制造 | 升降旋转系统、CNC加工、制造服务、工程设计 |
| 服务 | 住宅/商业/工业项目安装与维护 |

### 1.2 当前价值主张 (待优化)

**现有：**
> "Engineered Door and Window Gate Systems. Designed. Fabricated. Installed. Supported. Serving Industrial, Commercial and Residential."

**建议优化方向：**
> "Western Canada's fully integrated gate and door manufacturer. From Engineering to Installation – Everything Built In-House."

## 二、目标客户分析

### 2.1 客户细分

根据Digital Assessment报告，VanGATE的目标客户分为两大类：

| 客户类型 | 需求特征 | 决策因素 |
| --- | --- | --- |
| **住宅客户** | 车道门、车库门、门禁自动化 | 美观、安全、便利、价格 |
| **商业/工业客户** | 安全门、工业门、自动化系统 | 可靠性、定制能力、项目经验、认证资质 |

### 2.2 住宅客户画像

- **核心需求**：寻找 driveway gate、garage door
- **决策路径**：通常通过Google搜索"front gate"、"automatic gates"
- **关键诉求**：美观、耐用、自动化便利性

### 2.3 商业/工业客户画像

- **核心需求**：工业门、安全屏障、定制门禁解决方案
- **决策路径**：更长销售周期，需要案例和资质证明
- **关键诉求**：
  - 过往项目经验证明能力
  - 行业认证和资质
  - 定制能力
  - 维保响应能力

## 三、网站改版核心目标

### 3.1 战略目标

| 目标 | 优先级 | 衡量指标 |
| --- | --- | --- |
| 提升品牌专业形象 | 高 | 用户停留时间、跳出率 |
| 增加有机搜索流量 | 高 | 关键词排名、访问量 |
| 提升商业客户转化率 | 高 | 询盘表单提交、致电量 |
| 建立客户信任 | 高 | 案例研究阅读量、评价数量 |

### 3.2 网站改版关键要求

根据Digital Assessment报告，以下是关键改版要求：

#### 3.2.1 首页优化

| 编号 | 要求 | 当前状态 | 优先级 |
| --- | --- | --- | --- |
| HOM-001 | 明确价值主张：我是谁、我做什么、为什么选我 | 需优化 | P0 |
| HOM-002 | 添加above-the-fold内容展示核心优势 | 缺失 | P0 |
| HOM-003 | 添加客户logo展示 | 缺失 | P1 |
| HOM-004 | 添加认证和资质展示 | 缺失 | P1 |
| HOM-005 | 添加合作代理品牌展示 | 缺失 | P2 |
| HOM-006 | 添加项目数据统计 | 缺失 | P1 |

#### 3.2.2 导航结构优化

**问题诊断：**
- 当前导航混用产品/服务分类和客户类型分类，造成混淆
- 住宅客户不会搜索"Operator Controls"
- 商业客户不会搜索"Advanced CNC"

**建议导航结构：**

```
├── 首页
├── 解决方案
│   ├── 住宅
│   │   ├── 车道门
│   │   ├── 车库门
│   │   ├── 家庭电梯
│   │   ├── 门禁自动化
│   │   └── 维修服务
│   └── 商业/工业
│       ├── 安全门
│       ├── 工业门
│       ├── 屏障系统
│       ├── 门禁控制与自动化
│       └── 维保服务
├── 定制制造
│   ├── 升降旋转系统
│   ├── CNC加工
│   ├── 制造服务
│   └── 工程设计
├── 项目案例
│   ├── 住宅项目
│   ├── 商业项目
│   └── 工业项目
├── 关于我们
│   ├── 公司简介
│   ├── 为什么选择我们
│   ├── 生产设施
│   ├── 资质认证
│   └── 招聘
├── 资源
│   ├── 博客
│   ├── 常见问题
│   ├── 下载中心
│   └── 质保信息
└── 联系我们
```

#### 3.2.3 客户分群与信息架构

| 客户类型 | 信息需求 | 内容策略 |
| --- | --- | --- |
| 住宅客户 | 产品信息、报价请求、现场勘查预约 | 简洁产品信息，突出便利性 |
| 商业客户 | 案例研究、技术能力、资质证明 | 详细技术信息，突出专业性和经验 |

#### 3.2.4 行动号召 (CTA) 优化

**问题：** 当前CTA多为"Contact Us"，缺乏紧迫性和针对性。

**建议CTA：**

| 客户类型 | 建议CTA |
| --- | --- |
| 住宅客户 | Request a Quote / Book a Site Visit / Download Our Brochure |
| 商业客户 | Speak with a Technical Specialist / Request Project Consultation / Submit Drawings for Review |

## 四、内容策略要求

### 4.1 价值主张 (Why Us) 区块

建议在网站添加专门的"Why Us"区块，包含：

- [ ] 行业会员资格与认证
- [ ] 安全项目与认证
- [ ] 行业经验年限
- [ ] 公司愿景与价值观
- [ ] 团队文化
- [ ] 项目数量统计
- [ ] 客户Logo墙
- [ ] 客户评价

### 4.2 案例研究

**问题：** 缺乏针对商业客户的案例研究

**建议：** 开发5-10个强有力商业项目案例，每个包含：

- 项目名称
- 行业
- 项目范围
- 挑战
- 解决方案
- 项目照片

### 4.3 博客内容策略

**现状：** 博客内容5年以上未更新

**建议内容主题：**
- Swing vs Cantilever Gate 对比
- High Speed Door ROI 计算
- Commercial Gate Maintenance 指南
- Security Gate Requirements 规范
